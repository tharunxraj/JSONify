# JSONify — JSON Formatter & Visualizer

A simple developer tool I built to format and visualize messy JSON data instantly.

## Why I Built This

As a frontend developer, I kept seeing how raw JSON data is hard to read 
and debug. I wanted to build something that actually solves a real problem 
developers face every day. So I built JSONify from scratch using just HTML, 
CSS and JavaScript — no frameworks, no tutorials, just me figuring it out.

## What It Does

- Paste any messy JSON and instantly get a clean formatted output
- Detects errors and tells you exactly what is wrong
- Copy the formatted JSON with one click
- Minify JSON when you need compact data
- Switch between Dark and Light mode
- Works on all screen sizes

## Tech Stack

- HTML
- CSS
- JavaScript (Vanilla)
- Deployed on Netlify
